# Responsible AI Learning Map

A curated reference portal for exploring Responsible AI research — paper summaries, topic hubs, case studies, and source links. Built for personal reference and sharing with DBA cohort members.

## Live site

**https://sanjaygupta-professional.github.io/repoinsible-ai/**

## Open locally

Clone this repo and open `index.html` in any browser. No server, no login — just structure and resources.

## How it works

- **Topic hubs** — browse by area (transformers, alignment, fairness, …)
- **Paper library** — click any title to expand a short profile (what it is, why it matters, RAI angle), then **Read source** for arXiv or the official page
- **Quick index** — flat table for scanning
- **Case studies, debates, glossary** — cross-linked within the same file

## Optional: weekly arXiv discovery

```bash
bash weekly-digest.sh
```

Uses `arxiv-pp-cli` if installed. Output goes to `weekly/YYYY-MM-DD.json` — triage titles yourself; the portal does not ingest them automatically.
